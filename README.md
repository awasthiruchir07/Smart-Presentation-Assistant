# Smart Presentation Assistant

## Problem
Students often struggle to organize their thoughts and manage time during presentations. They forget key points and lack structured speaking flow.

## Solution
This project is a Command Line Interface (CLI) tool that helps users create structured presentations, generate speaking prompts, and practice with a timer.

## Features
- Create presentations with multiple slides
- Automatically generate speaking prompts
- View structured notes
- Practice presentations with timer
- Delete presentations
- Data stored in CSV format

## Tech Stack
- Python
- CSV file handling

## How to Run

1. Clone the repository
2. Navigate to project folder
3. Run:
    bash python app.py

## Example Usage
- Create a presentation
- Add slide titles
- View prompts
- Practice using timer
